# Usage

## Preferred mode

Use the skills to review existing audit artifacts:

- structured JSON
- Markdown report
- compact summary

The skills should:

- separate sitewide blockers from page-level evidence
- collapse duplicate findings into one action area
- link every accessible artifact path they reference
- avoid inventing heuristics outside the shared audit contract

## Optional mode

If the public CLI is already available in the environment, the skills may use it
to generate fresh artifacts before reviewing them. The skills themselves remain
dependency-free.
